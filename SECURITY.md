# Security Policy

## Supported versions

PatchPact is still early-stage, so the latest `main` branch is the primary supported version.

## Reporting a vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Instead, report them privately to the maintainer account associated with this repository:

- GitHub: [@FlowerNeverFade](https://github.com/FlowerNeverFade)

When possible, include:

- a description of the vulnerability
- affected files or surfaces
- reproduction steps or proof of concept
- potential impact
- any suggested mitigation

We will do our best to acknowledge reports quickly and coordinate a fix before public disclosure.
